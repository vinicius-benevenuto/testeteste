O som do sucesso é o silêncio.”  
Porque uma tradução perfeita é aquela que ninguém percebe que existe.  
É aquela compreendida sem esforço.  
É aquela que envolve a todos, levando a mensagem — e somente a mensagem.  
Sem ruídos, sem atrasos, sem distrações.

O silêncio de quem entendeu tudo na primeira vez  
é exatamente o sucesso do qual estamos falando.

E, se você já sentiu a diferença que um evento sem ruídos faz,  
sabe o poder desse silêncio.
